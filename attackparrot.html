Attack Parrot

We set out to make something we would want to play with.

Using Pupil Labs software, Arduino, and 3D printing we made an eye controlled rubberband shooter.


Eye Movement

We used Pupil, a hardware and software eye tracking system, to receive coordinate input of where to fire.Pupil has a large library of source code for tracking pupils. We used their open source code but used a specific data transform to get out input for an Arduino.


The Turret

We made a custom 3D printed turret powered by servos. We had a servo to control the x and y-axis as well as firing.


The Connection

We used an Arduino Uno to take inputs from the Pupil software and drive the servos to specific locations. This allowed aiming and firing.
